Getting Started

Everything you need to know to integrate with Craft APIs quickly and securely.

1. Authentication

Craft uses JSON Web Tokens (JWT) for API authentication. You must include the token in the `Authorization` header of all your API requests.

Authorization: Bearer <your_access_token>

2. Environments

Development / Sandbox

http://127.0.0.1:8000

Production Coming Soon

https://api.craft.com

3. Making your first request

Here is an example using `cURL` to fetch the health status of the Admin API:

curl -X GET \
  'http://127.0.0.1:8000/admin-api/health/' \
  -H 'Authorization: Bearer YOUR_TOKEN_HERE' \
  -H 'Accept: application/json'
